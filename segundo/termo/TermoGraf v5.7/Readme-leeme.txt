Welcome to TermoGraf / Bienvenido a TermoGraf

----------------------------------------------------------------
ENGLISH
You can run TermoGraf in any Operative System with Java 1.5 or superior. You can download the official Java version in http://www.java.com.
To run TermoGraf there are two options:
1.- Only for MsWindows systems: double-click the "TermoGraf (Windows).exe" file.
2.- Only for Linux/MacOS systems: edit the "TermoGraf (Linux-MacOS).sh" file, modify it according its instructions and execute it.
3.- For any Operative System: double-click the "termograf.jar" file (inside the "bin" folder). Note: If it is not properly opened, click on this file with the right button of the mouse, select the "Open with" menu and choose "Java". This opening maight be configured on the operative system following these instruccions: 1) MsWindows: "Open with / Choose program" menu y check the "Always use the selected program" option. 2) Linux/Macintosh: "Properties" menu, "Open wih" tab.

Any sugestion is welcome at: info.termograf@unizar.es

ESPAÑOL
TermoGraf se puede ejecutar en cualquier Sistema Operativo con Java 1.5 o superior. Se puede descargar la versión oficial de Java en http://www.java.com.
Para abrir TermoGraf Pro existen dos opciones:
1.- Sólo para sistemas MsWindows: doble-clic sobre el fichero "TermoGraf (Windows).exe".
2.- Sólo para sistemas Linux/MacOS: editar el fichero "TermoGraf (Linux-MacOS).sh", modificarlo según sus instrucciones y ejecutarlo.
3.- Para cualquier sistema operativo: doble-clic sobre el fichero "termograf.jar" (dentro de la carpeta "bin"). Nota: Si no se abre correctamente, pulsar sobre este fichero con el botón derecho del ratón, seleccionar el menú "Abrir con" y elegir "Java". Esta forma de apertura se puede configurar en el sistema operativo mediante las siguientes instrucciones: 1) MsWindows: menú "Abrir con / Elegir programa" y marcando la casilla "Utilizar siempre este programa", 2) Linux/Macintosh: menú "Propiedades", solapa "Abrir con".

Cualquier sugerencia es bien recibida en: info.termograf@unizar.es
