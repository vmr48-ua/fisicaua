Ventana "Enunciado"

Esta ventana muestra el enunciado a resolver para el documento de TermoGraf, que variará de un alumno a otro en caso de ser un documento multiejercicio. Las preguntas que se formulan en esta ventana se contestan en el panel "Respuestas".

Ventana "Enunciado"

Contenido

Panel "Enunciado"

Este panel muestra el enunciado que debe resolver el estudiante. La edición del enunciado se realiza en la ventana "Editar enunciado " de la versión TermoGraf Pro.

 

Panel "Datos del estudiante"

Este panel muestra los datos del estudiante que está resolviendo el ejercicio. Mediante el botón "Editar estudiante" se pueden modificar estos datos si alguno está incompleto, excepto la clave del estudiante en caso de ser un multiejercicio.

 

Panel "Respuestas"

En la parte superior de este panel aparece el nombre del estudiante que está resolviendo ejercicio. Sus datos completos aparecen pulsando el borón "Información".

A continuación se muestran las respuestas a suministrar para este ejercicio. El botón "Ver soluciones" aparece en caso de que este ejercicio permita la auto-corrección, pudiendo configurarse esta opción en la ventana "Configuración de ejercicio" de la versión TermoGraf Pro.

 

Panel "Comentario"

En este panel se pueden escribir comentarios adicionales a las respuestas.

La opción "Se adjuntan otros documentos" informa al profesor de que se van a entregar documentos adicionales junto con este ejercicio TermoGraf.
