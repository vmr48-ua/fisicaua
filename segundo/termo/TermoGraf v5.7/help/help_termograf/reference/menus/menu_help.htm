Menú "Ayuda"

Este menú muestra información sobre TermoGraf y cómo ponerse en contacto con los autores. Para poder realizarse varias de estas acciones es necesario que el navegador de TermoGraf esté bien configurado en la ventana "Preferencias iniciales".

Menú "Ayuda"

Abre el navegador del sistema operativo con los documentos de información dentro del ordenador acerca del funcionamiento de TermoGraf.

 

Menú "Tutorial"

Abre el documento de información acerca del funcionamiento básico de TermoGraf.

 

Menú "Ayuda (Web)"

Abre el navegador del sistema operativo con la dirección de la página web correspondiente a la ayuda de TermoGraf. Esta página ofrece información actualizada de las últimas funciones añadidas al programa.

 

Menú "TermoGraf (Web)"

Abre el navegador del sistema operativo con la dirección de la página web de TermoGraf.

 

Menú "Enviar errores"

Abre la ventana "Enviar errores" para mandar un correo electrónico anónimo con información acerca de errores encontrados en TermoGraf.

Ventana "Enviar errores"

Esta ventana tiene los siguientes apartados:

  • Para: Dirección de soporte técnico de TermoGraf. Este campo no es editable.
  • Asunto: Título del mensaje. Este campo no es editable.
  • Adjuntar: Documento opcional para enviar con el e-mail.
  • Opción "Adjuntar documento actual de TermoGraf": Envía una copia del documento con el que se está trabajando. Se utiliza para poder reproducir el error en las mismas condiciones originales.
  • Mensaje: Información del usuario acerca del error. Un error en el programa no puede ser corregido si no es posible reproducirlo, por lo que se recomienda enviar la información más concreta posible de los pasos que se han dado para producir el error.
  • Error: Información del funcionamiento de TermoGraf en la última sesión. Las primeras líneas informan sobre el contexto informático en el que trabaja el programa.

 

Menú "Acerca de TermoGraf"

Abre la ventana "Acerca de TermoGraf" con información sobre la versión que se está utilizando y los autores de TermoGraf.
