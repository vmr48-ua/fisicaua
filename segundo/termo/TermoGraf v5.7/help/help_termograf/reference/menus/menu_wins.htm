Menú "Ventanas"

Este menú contiene las ventanas auxiliares de TermoGraf.

Menú "Tabla"

Abre la ventana "Tabla" para crear tablas de datos con los valores de los elementos termodinámicos y de las variables de usuario. Los datos se pueden manipular para mostrarlos en una tabla de datos, representarlos en las gráficas inferiores de la ventana o guardarlos como texto para ser utilizados en otro programa. El contenido de esta ventana también se puede imprimir mediante el menú "Imprimir/Guardar imagen".

 

Menú "Editor de ecuaciones"

Abre la ventana "Editor de ecuaciones" para crear ecuaciones con las propiedades termodinámicas de los estados, procesos y ciclos introducidos en el diagrama termodinámico. Las ecuaciones creadas se pueden almacenar como variables para ser manipuladas desde la ventana "Variables" dentro del todo el documento.

 

Menú "Variables"

Abre la ventana "Variables" en donde se gestionan las variables creadas por el usuario. Las variables se utilizan para escribir ecuaciones no contempladas en el programa y obtener su resultado, tablas de datos, gráficas o para utilizarlas en la elaboración de un documento multiejercicio.

 

Menú "Enunciado"

Abre la ventana "Enunciado" en donde se muestra el enunciado y las preguntas a resolver para el documento de TermoGraf, que variarán de un alumno a otro en caso de ser un documento multiejercicio. La edición del enunciado se realiza en la ventana "Editar enunciado " de la versión TermoGraf Pro.

 

Menú "Enviar e-mail"

Abre la ventana "Enviar e-mail" desde donde se puede enviar un correo electrónico desde TermoGraf. Está pensada para que los estudiantes puedan enviar sus dudas o ejercicios resueltos al profesor sin depender del ordenador desde el que estén trabajando, ya que podría tener un programa de correo de difícil configuración o no tenerlo instalado.

 

Menú "Conexión FTP"

Abre la ventana "Conexión FTP" desde donde se envian y reciben documentos mediante el estándar FTP -File Transfer Protocol-, ampliamente utilizado en la red para el intercambio de ficheros. Está pensada para facilitar la comunicación entre alumno y profesor habilitando un espacio común donde el profesor pueda dejar los ejercicios y que el alumno los tenga disponibles cuando vaya a trabajarlos. Con esta ventana se evita depender de que el ordenador donde se esté trabajando disponga de un programa de FTP con una difícil configuración o no tenerlo instalado.

 

Menú"Editor de ciclos" [TermoGraf Pro]

Abre la ventana "Editor de ciclos" en donde se facilita la creación de ciclos de uso frecuente en el desarrollo de una asignatura de termodinámica. Esta ventana sólo está disponible para la versión TermoGraf Pro.

 

Menú"Editar enunciado" [TermoGraf Pro]

Abre la ventana "Editar enunciado" en donde se edita el contenido del enunciado del ejercicio. Esta ventana sólo está disponible para la versión TermoGraf Pro.

 

Menú"Multiejercicio" [TermoGraf Pro]

Abre la ventana "Multiejercicio" en donde se gestiona la creación de un documento multiejercicio. Los documentos multiejercicio contienen variantes de un mismo enunciado de forma que a diferentes estudiantes les correspondan diferentes preguntas. Esta ventana sólo está disponible para la versión TermoGraf Pro.

 

Menú"Corrección" [TermoGraf Pro]

Abre la ventana "Corrección" en donde se facilita la corrección de los documentos multiejercicio y la presentación de los resultados obtenidos por los estudiantes. Esta ventana sólo está disponible para la versión TermoGraf Pro.
